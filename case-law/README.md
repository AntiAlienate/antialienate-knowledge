# Case Law

Court rulings on parental alienation, related custody decisions, and applicable jurisprudence — catalogued by jurisdiction.

## Coverage targets

| Jurisdiction | Target entries | Priority |
|---|---:|---|
| [ECHR](./echr) (Article 8 family-life cases) | 30+ | High — ECHR rulings bind 46 member states |
| [Belgium](./belgium) (Civil Code + Cour de Cassation + Tribunal de la Famille) | 50+ | High — home jurisdiction, complete coverage |
| [United States](./united-states) (federal + 50 states) | 500+ | High — largest English-speaking market |
| [United Kingdom](./united-kingdom) (Family Court + Court of Appeal) | 30+ | High — Re S [2020], Re C [2023] |
| [Australia](./australia) (Family Law Act 60CC + cases) | 30+ | Medium |
| [Canada](./canada) | 20+ | Medium |
| [Brazil](./brazil) (Law 12.318/2010 — world's most progressive PA legislation) | 30+ | Medium |
| [France](./france) (Cour de Cassation) | 15+ | Medium |
| [Germany](./germany) (BGH Familienrecht) | 15+ | Medium |
| [Italy](./italy) (Bondavalli, Improta line) | 20+ | Medium |
| [Netherlands](./netherlands) | 15+ | Medium |

## Entry format

Each case file follows:

```
# {Case name} — {Court}, {Year}
**Citation**: {full citation}
**Court**: {court name + chamber}
**Date**: {YYYY-MM-DD}
**Parties**: {anonymized if needed}
**Source**: {link to ruling}

## Holding
{one-paragraph summary of the legal holding}

## Facts
{relevant facts of the case}

## Reasoning
{court's reasoning, key paragraphs}

## Significance for parental alienation
{how this case supports / shapes PA jurisprudence}

## Cites
- {cases cited within the ruling that are also in this repo}

## Cited by
- {subsequent cases that cite this one}

---
Last reviewed: YYYY-MM-DD by {contributor}.
License: CC BY 4.0.
```
