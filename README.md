# AntiAlienate — Open Knowledge Base

> The most comprehensive open-source reference on parental alienation: statutes by jurisdiction, current case law, peer-reviewed research, practical templates, and shareable infographics. **Free. CC BY 4.0. No paywall.**

If you are a parent dealing with parental alienation, this repository exists so you don't have to start from scratch. If you are a practitioner, advocate, or researcher, this is structured reference material you can cite, adapt, and contribute to.

## Start here

**If you've just been told you're losing contact with your child** → [/templates/](./templates/) for the artefacts you need this week, then [/jurisdictions/](./jurisdictions/) for your jurisdiction's legal framework.

**If you're preparing for a court hearing** → [/templates/court-bundle-structure.md](./templates/court-bundle-structure.md), [/templates/cafcass-prep-brief.md](./templates/cafcass-prep-brief.md), and the [/landmark-cases/](./landmark-cases/) you should know.

**If you need an evidence-gathering tool right now** → [/templates/gdpr-art-15-school-sar.md](./templates/gdpr-art-15-school-sar.md) and [/templates/contact-log-spreadsheet.md](./templates/contact-log-spreadsheet.md).

**If you want shareable educational material** → [/open-source/infographics/](./open-source/infographics/) — 17 purpose-built reference cards covering frameworks, statistics, vocabulary, and tactics.

**If you're researching the field** → [/research/](./research/) for foundational works, [/case-law/2024-2026-rulings.md](./case-law/2024-2026-rulings.md) for the cross-jurisdiction recent-rulings consolidation, [/tools/legal-research.md](./tools/legal-research.md) for the 12 free authoritative databases to verify against.

## How this is maintained

A background agent polls four public legal-research sources every 25 minutes (PubMed, CrossRef, CourtListener, BAILII), dedupes against a known-finds registry, and pushes new material into [/digest/](./digest/) automatically. Memes, value cards, jurisdiction pages, and templates are added on a rolling basis. The repo grows continuously.

## Verified Upstream Publishers

We credit independent publishers whose work we mirror or draw from. Crediting upstream sources openly is how a small field grows into a serious reference.
